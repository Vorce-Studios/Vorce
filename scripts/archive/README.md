# Archive

This archive keeps scripts and log artifacts that are currently not trusted enough to delete
but also not trusted enough to keep as active tooling.

## Review

`review/` contains scripts whose ongoing use is unclear. They are kept here until it becomes clear
whether something still depends on them.

## Entrypoints

`entrypoints/` contains former top-level wrapper scripts. They are archived, not deleted.

## Legacy

`legacy/` contains superseded active tooling that should no longer be used for new workflows.

## Logs

`logs/` contains historical log files that do not belong in active script folders but are kept
until deletion is explicitly approved.
