# DOC-C3: Hardware Integration

## 1. MIDI & OSC

* **MIDI**: Unterstützung für Controller via `midir`. "Midi Learn" Funktion für UI-Elemente.
* **OSC**: Fernsteuerung via Netzwerk-Paketen. Adressierbar über konfigurierbare Pfade.

## 2. Philips Hue (Entertainment API)

* **DTLS**: Nutzt `webrtc-dtls` für verschlüsseltes Streaming in Echtzeit.
* **Lichter**: Abbildung von Layer-Farben auf smarte Beleuchtung im Raum.
