# API Reference

This directory is reserved for API documentation and reference materials.

## Rust API Documentation

MapFlow uses standard Rust tools for API documentation. To generate and view the API documentation locally, run:

```bash
cargo doc --workspace --no-deps --open
```

This will generate the documentation for all crates in the workspace and open it in your default browser.

## Hosted Documentation

Hosted documentation is currently not available. Please generate the documentation locally using the command above.
