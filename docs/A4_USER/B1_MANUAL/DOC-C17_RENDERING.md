# Rendering und Export

**Was es macht:**
Vorce bietet einen dedizierten Render-Dialog, um deine Visuals und Mappings als Video-Dateien zu exportieren. Hierbei kannst du spezifische GPU-Settings für optimale Performance wählen.

![Screenshot: Der Render-Dialog mit ausgeklappten GPU-Settings](docs/assets/missing/export-dialog-gpu.png)

**So benutzt du es:**

1. Gehe im Hauptmenü auf **File** > **Export**.
2. Im Render-Dialog wählst du das gewünschte Format (z.B. MP4).
3. Klappe die **GPU-Settings** aus und wähle deinen bevorzugten Hardware-Encoder (z.B. NVENC oder AMF).
4. Klicke auf **Start Render**.
